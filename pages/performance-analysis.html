Robustness Evaluation

Overall Model Robustness Ranking

Overall Robustness Ranking Methodology

The overall robustness ranking is computed using a ranking-based scoring system based on Area Under the Curve (AUC) of TPR@N%FPR-PSNR curves across all attack types.

AUC Calculation for TPR@N%FPR-PSNR Curves:

For each model and attack type, we compute the AUC of the TPR@N%FPR vs. normalized PSNR curve:

PSNRnorm = (PSNR - PSNRmin) / (PSNRmax - PSNRmin)

AUC = ∫01 TPR@N%FPR(PSNRnorm) d(PSNRnorm)

The curve includes baseline points: (0, 0) representing no attack quality and (1, TPR@N%FPRNoAttack) representing clean image performance. This provides a comprehensive measure of robustness across different attack intensities.

Ranking-Based Scoring:

For each attack type, models are ranked by their AUC scores. Points are assigned as:

Points = N - rank + 1, where N = total number of models, rank ∈ [1, N]

The overall score is the sum of ranking points across all attack types. Higher total score indicates better overall robustness performance.

Robustness Scores under Different Attacks

Robustness Scores Methodology

The robustness scores are computed using Area Under the Curve (AUC) of TPR@N%FPR-PSNR curves for each specific attack type, providing a comprehensive measure of model performance across different attack intensities.

AUC Calculation Process:

For each model-attack pair, we extract TPR@N%FPR and PSNR data points through the following steps:

1. Extract TPR@N%FPR values across all attack factors

2. Extract corresponding PSNR values from visual quality data

3. Normalize PSNR: PSNRnorm = (PSNR - PSNRmin) / (PSNRmax - PSNRmin)

4. Create TPR@N%FPR-PSNR curve with baseline points: (0,0) and (1, TPR@N%FPRNoAttack)

5. Calculate AUC = ∫01 TPR@N%FPR(PSNRnorm) d(PSNRnorm)

TPR@N%FPR: True Positive Rate at N% False Positive Rate, measuring watermark detection accuracy under attack.

Global PSNR Range: Determined across all models and attacks for consistent normalization. If max PSNR < 60dB, assumes 60dB as "No Attack" reference; otherwise uses actual max + 10dB.

Ranking System:

Models are ranked by AUC scores for each attack type, with higher AUC indicating better robustness (maintaining high TPR@N%FPR across various attack intensities while preserving visual quality).

Robustness Curves under Different Attacks

Robustness Curves Analysis

Robustness curves visualize model performance degradation across varying attack intensities, enabling analysis of robustness tolerance and attack vulnerability patterns.

Chart Type Selection:

Multi-factor attacks: Line plots with markers where X-axis shows attack factors sorted numerically, Y-axis shows performance metrics (TPR@N%FPR, Extract Accuracy), and multiple lines compare different models.

Single-factor attacks: Bar charts where X-axis shows model names, Y-axis shows single performance value, with different colors for each model.

Factor Processing:

Attack factors are converted to numeric values and sorted for consistent visualization.

Performance Metrics:

TPR@N%FPR: True Positive Rate at N% False Positive Rate (range: [0,1])

Extract Accuracy: Watermark extraction accuracy (range: [0,100%])

Curves help identify attack tolerance thresholds and compare relative robustness across different attack types and intensities.

Attack Effectiveness Analysis

Attack Effectiveness Ranking

Attack Effectiveness Ranking Methodology

Attack effectiveness is measured using the AE (Attack Effectiveness) metric, which combines attack impact on watermark detection with visual quality preservation.

AE Metric Definition:

AE = (1 - TPR@N%FPR) × PSNRnormalized

Where (1 - TPR@N%FPR) represents attack impact (higher = more detection degradation), PSNRnormalized represents visual quality preservation (higher = better quality), and PSNRnorm = (PSNR - PSNRmin) / (PSNRmax - PSNRmin).

Calculation Process:

1. Global PSNR Range: Extract all PSNR values across models/attacks

2. Per-Factor AE: Compute AE for each (model, attack, factor) combination

3. Attack Average: Average AE across all models and factors for each attack

4. Ranking: Sort attacks by average AE (descending = more effective)

Interpretation:

Higher AE indicates more effective attacks that successfully reduce watermark detectability while maintaining reasonable visual quality. The ranking helps identify which attack types pose the greatest threat to watermark systems.

Example: An attack with AE=0.8 significantly degrades detection (low TPR@N%FPR) while preserving high image quality (high PSNR), making it highly effective.

Model Robustness Curves under Different Attacks

Model vs. Attacker Performance Analysis

Detailed analysis of attacker performance curves showing the trade-off between visual quality (PSNR) and robustness degradation for each model under different attack types.

Attacker Performance Curve:

For each model and combination of visual quality and robustness metrics, we generate curves showing how much visual quality attackers sacrifice to achieve robustness degradation.

Curve Specification:

X-axis: Visual Quality Metric (PSNR, excludes SSIM)

Y-axis: Robustness Metric (TPR@N%FPR, Extract Accuracy)

Lines: Different attack types (excludes "No Attacking")

Points: Different attack factor intensities

Analysis Dimensions:

1. Trade-off Analysis: How attack intensity affects quality vs. robustness

2. Attack Comparison: Which attacks are most efficient (high robustness impact, low quality loss)

3. Model Vulnerability: Identify weak points in different watermark models

4. Factor Sensitivity: Model response to varying attack intensities

Data Processing:

For each attack factor, we extract both robustness and visual quality metrics, then sort by visual quality values to create proper curve visualization. Points are connected to show the performance degradation trajectory as attack intensity increases.

Interpretation: Steeper curves indicate higher attack efficiency, while flatter curves suggest better model resilience to that attack type.
